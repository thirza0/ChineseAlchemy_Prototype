流程
診斷資訊
問診系統
問診系統可得知某病患的
- 體質五行:以五行相生、相剋兩種系統判斷毒性與效果是否有額外加成
- 主要需求/次要需求:病患會有主要想治好的症狀與其他症狀，丹藥能夠滿足病患的越多症狀滿意度越高(目前丹藥的品質不會有影響)
- 毒素累積:病患體內可能已經累積一定的毒素，如果給予的丹藥累加會超過病患能承受的上限，病患會死亡
煉丹流程
根據病患的資訊，玩家要判斷要練出什麼藥。
最一開始玩家須要有明確知道配方名稱與症狀(至少一種)的對應，才會知道要以哪個配方為目標。

閒置
畫面中有三個流派按鈕可選擇
不同流派的計算差異只在土的應用方式。
     中和流:土作為中心點，無論土算出來的距離是多少，都會乘以原點向量(0,0)，即為土的材料加得越多，座標會越靠近中心點
     延伸流:土作為加成材料，土會無條件依循另一材料屬性的方向，即為土的材料是一種增強藥性的材料
     偏性流:土作為調和的材料，即為左方算法，土會去補足另一軸的強度
*建議後續以偏性流為主，可以使配方座標變化較大
